Introducción a Git

Comandos Básicos

  • git init: Inicializar un repositorio
  • git status: Verifica el estado de mis archivo
  • git add file_name: Guarda el archivo el área de preparado
  • git commit -m "message": Crea una versión de nuestra aplicación
  • git log: Muestra las versiones de mi aplicación

Branching

  • git branch: Muestra las ramas de mi aplicación
  • git branch new_branch_name: Crea una nueva rama de mi aplicación
  • git checkout branch_name: Permite moverme entre mis ramas
  • git brach_name -d: Eliminar mi branch
  • git merge branch_name: Une dos ramas de mi aplicación
